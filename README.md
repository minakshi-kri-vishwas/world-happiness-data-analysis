
# 🌍 World Happiness Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the World Happiness Dataset using Python. It analyzes various factors affecting happiness scores across different countries and regions and presents insights through statistical analysis and visualizations.

## 🚀 Features

* Data Cleaning and Preprocessing
* Handling Missing Values
* Country-wise and Region-wise Analysis
* Happiness Level Classification
* Correlation Analysis
* Statistical Analysis using NumPy
* Data Visualization with Matplotlib

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab / Jupyter Notebook

## 📊 Analysis Performed

* Top 10 Happiest Countries
* Bottom 10 Least Happy Countries
* GDP vs Happiness Score Analysis
* Freedom vs Happiness Score Analysis
* Regional Happiness Comparison
* Distribution of Happiness Scores

## ▶️ Installation

1. Clone the repository:
   git clone https://github.com/yourusername/world-happiness-data-analysis.git

2. Move to the project directory:
   cd world-happiness-data-analysis

3. Install dependencies:
   pip install -r requirements.txt

4. Run the notebook using Jupyter Notebook or Google Colab.
